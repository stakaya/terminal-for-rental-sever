概要
    WindowsクライアントからWindowsやUnix等のサーバを操作するソフトです。

機能
    SSHを利用できないレンタルサーバで、SSHライクな操作が行えます。
    ファイルのリネームや削除、コマンドの実行などが行えます。
    Webサイト運用ツールとしてご利用いただけます。

システム要件
    * クライアントはWindowsNT4.0〜Windows7で動作します。
    * サーバはPHPが動作するWebサーバが必要です。
    * PHPは5.0以降が必要です。

インストール
    PHPが動作するWebサーバ上のディレクトリに、
    (Apacheではhtdocs、IISではwwwroot等の配下)
    下記のファイルをアップロードします。

      server.php

    下記のファイルをクライアントPCの適当な場所へ配置します。

      client.exe

アンインストール
    Webサーバにアップロードしたphpファイルを削除します。
    クライアントのexeファイルを削除します。

操作説明
    クライアントに配置したexeファイルをコマンドプロンプトから起動します。
    コマンドライン引数には接続ホスト名、ファイル名、パスコードを指定します。
    起動時に毎回引数を指定するのは面倒ですので、バッチファイルを作成するか、
    exeファイルのショートカットを作成して引数を指定することをお勧めします。

    例 c:\>client.exe localhost server.php passcode

    ※パスコードの設定場所はserver.phpの2行目に設定します。
